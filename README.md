# Explainable Lead Scoring App

This project helps sales teams prioritize leads using a smart scoring system.

## Features
- Adjustable weights (industry, location, contactability, etc.)
- Explainable lead scoring
- CSV upload option
- Smart recommendations

## How to run

pip install -r requirements.txt
streamlit run app.py
